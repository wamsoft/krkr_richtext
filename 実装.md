# 実装進捗（吉里吉里プラグイン）

## ビルド方法

```bash
# 事前準備: VCPKG_ROOT 環境変数を設定

# CMake 構成（初回または CMakeLists.txt 変更時）
make prebuild                      # OS 自動検出
make prebuild PRESET=x64-windows   # プリセット明示指定

# ビルド
make build
make build BUILD_TYPE=Debug        # デバッグビルド
```

## 依存関係

- `../richtext` — コアライブラリ（richtext_lib）
- `../tp_stub` — 吉里吉里プラグインSDK
- `../ncbind` — C++ ↔ TJS2 バインダー

## TJS バインディング — 進行中

- main.cpp — ncbind によるプラグインインタフェース実装

### 公開クラス

| TJS クラス | 内部クラス | 説明 |
|-----------|-----------|------|
| RichTextStyle | TextStyle ラッパー | フォント・サイズ・スタイル設定 |
| RichTextAppearance | Appearance ラッパー | 塗り・ストローク・影設定 |
| RichTextLayout | TextLayout ラッパー | 1行レイアウト結果 |
| RichTextParagraphLayout | ParagraphLayout ラッパー | 複数行レイアウト結果 |
| RichText | 静的クラス | フォント登録・定数 |
| LayerExRichText | Layer 拡張 | 描画メソッド群 |

### 主要メソッド（Layer 拡張）

- `drawStyleText` — 1行テキスト描画
- `drawStyleParagraph` — パラグラフ描画
- `drawStyleTaggedText` — タグ付きテキスト描画
- `drawStyleParagraphLayout` — 事前計算レイアウトの描画（maxGlyphs 対応）
- `fillRect` — 矩形描画
- `measureStyleText` / `measureStyleParagraph` — 計測
- `sync` — thorvg 描画同期
- `reset` — キャンバスバッファ更新
